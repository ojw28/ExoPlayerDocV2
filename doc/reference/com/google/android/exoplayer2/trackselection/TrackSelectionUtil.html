        Returns bitrate values for a set of tracks whose formats are given, using the given upcoming media chunk iterators and the queue of already buffered MediaChunks.
        Parameters:
        formats - The track formats.
        queue - The queue of already buffered MediaChunks. Must not be modified.
        maxPastDurationUs - Maximum duration of past chunks to be included in average bitrate values, in microseconds.
        iterators - An array of MediaChunkIterators providing information about the sequence of upcoming media chunks for each track.
        maxFutureDurationUs - Maximum duration of future chunks to be included in average bitrate values, in microseconds.
        useFormatBitrateAsLowerBound - Whether to return the estimated bitrate only if it's higher than the bitrate of the track's format.
        bitrates - An array into which the bitrate values will be written. If non-null, this array is the one that will be returned.
        Returns:
        Bitrate values for the tracks. As long as the format of a track has set bitrate, a bitrate value is set in the returned array. Otherwise it might be set to Format.NO_VALUE.
