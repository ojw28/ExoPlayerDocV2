        Creates load event info.
        Parameters:
        dataSpec - Defines the requested data.
        uri - The Uri from which data is being read. The uri must be identical to the one in dataSpec.uri unless redirection has occurred. If redirection has occurred, this is the uri after redirection.
        responseHeaders - The response headers associated with the load, or an empty map if unavailable.
        elapsedRealtimeMs - The value of SystemClock.elapsedRealtime() at the time of the load event.
        loadDurationMs - The duration of the load up to the event time.
        bytesLoaded - The number of bytes that were loaded up to the event time. For compressed network responses, this is the decompressed size.
