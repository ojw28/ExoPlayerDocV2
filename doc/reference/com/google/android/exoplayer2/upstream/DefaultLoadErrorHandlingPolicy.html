      • getRetryDelayMsFor

        public long getRetryDelayMsFor(int dataType,
                               long loadDurationMs,
                               IOException exception,
                               int errorCount)
        Retries for any exception that is not a subclass of ParserException, FileNotFoundException or Loader.UnexpectedLoaderException. The retry delay is calculated as Math.min((errorCount - 1) * 1000, 5000).
        Specified by:
        getRetryDelayMsFor in interface LoadErrorHandlingPolicy
        Parameters:
        dataType - One of the C.DATA_TYPE_* constants indicating the type of data to load.
        loadDurationMs - The duration in milliseconds of the load up to the point at which the error occurred, including any previous attempts.
        exception - The load error.
        errorCount - The number of errors this load has encountered, including this one.
        Returns:
        The number of milliseconds to wait before attempting the load again, or C.TIME_UNSET if the error is fatal and should not be retried.
