Annotation Type DataSpec.Flags


  • @Documented
 @Retention(value=SOURCE)
public static @interface DataSpec.Flags
    The flags that apply to any request for data. Possible flag values are DataSpec.FLAG_ALLOW_GZIP, DataSpec.FLAG_ALLOW_ICY_METADATA, DataSpec.FLAG_DONT_CACHE_IF_LENGTH_UNKNOWN and DataSpec.FLAG_ALLOW_CACHE_FRAGMENTATION.
