com.google.android.exoplayer2.offline

Class DownloadManager

  • java.lang.Object
    • com.google.android.exoplayer2.offline.DownloadManager

  • public final class DownloadManager
extends Object
    Manages multiple stream download and remove requests.

    A download manager instance must be accessed only from the thread that created it, unless that thread does not have a Looper. In that case, it must be accessed only from the application's main thread. Registered listeners will be called on the same thread.
