      • move

        void move(int indexFrom,
          int indexTo)
        Moves an existing item within the playlist.

        Calling this method is equivalent to removing the item at position indexFrom and immediately inserting it at position indexTo. If the moved item is being played at the moment of the invocation, playback will stick with the moved item.

        Parameters:
        indexFrom - The index of the item to move.
        indexTo - The index at which the item will be placed after this operation.
        Throws:
        IndexOutOfBoundsException - If for either index, index < 0 || index >= getSize().

      • removeRange

        void removeRange(int from,
                 int exclusiveTo)
        Removes a range of items from the queue.

        Does nothing if an empty range (from == exclusiveTo) is passed.

        Parameters:
        from - The inclusive index at which the range to remove starts.
        exclusiveTo - The exclusive index at which the range to remove ends.
        Throws:
        IndexOutOfBoundsException - If from < 0 || exclusiveTo > getSize() || from > exclusiveTo.
