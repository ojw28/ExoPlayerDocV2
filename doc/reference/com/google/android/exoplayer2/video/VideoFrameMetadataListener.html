      • onVideoFrameAboutToBeRendered

        void onVideoFrameAboutToBeRendered(long presentationTimeUs,
                                   long releaseTimeNs,
                                   Format format)
        Called when the video frame about to be rendered. This method is called on the playback thread.
        Parameters:
        presentationTimeUs - The presentation time of the output buffer, in microseconds.
        releaseTimeNs - The wallclock time at which the frame should be displayed, in nanoseconds. If the platform API version of the device is less than 21, then this is the best effort.
        format - The format associated with the frame.
